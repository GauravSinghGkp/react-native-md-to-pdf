# react-native-md-to-pdf

Convert Markdown to PDF in React Native apps

## Installation


```sh
npm install react-native-md-to-pdf
```


## Usage


```js
import { multiply } from 'react-native-md-to-pdf';

// ...

const result = await multiply(3, 7);
```
